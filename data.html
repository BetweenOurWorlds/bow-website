Data

The data is available by downloading an RDF datadump or by quering our Triple Pattern Framgents server.

Datadump

The datadump is here available in the N-Triples format.

Triple Pattern Fragments Server

The data can be queried via a Triple Pattern Fragments server. The URL of the server is http://needtoupdatethis.com. Psssst! Visit the URL in the browser to have a quick look at the data! You can query the data by requesting a single fragment or by using SPARQL queries, which combines multiple fragments for you. Several libraries in different programming languages are available that help you with that.
