Data

The data is available by downloading an RDF datadump or by querying our Triple Pattern Fragments server. Existing ontologies are used as much as possible, but for some concepts we developed our own ontology. We schedule a new release of the dataset every month, which includes the updates of the metadata provided by the Kitsu API.

Triple Pattern Fragments Server

The data can be queried via a Triple Pattern Fragments server. The URL of the server is {{site.tpfurl}}. Psssst! Visit the URL in the browser to have a quick look at the data! You can query the data by requesting a single fragment or by using SPARQL queries, which combines multiple fragments for you. Several libraries in different programming languages are available that help you with that.
