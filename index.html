A Linked Open Dataset with the focus on Anime

Between Our Worlds is an initiative to provide metadata information about Anime as Linked Open Data. 'Wait, what is this Linked Data thingy you speak of?' Linked Data is a principle that ++++ The core contribution of our initiative a dataset, which is accessible via several human and machine-understandable methods.

The dataset has as goal to

  • Discoverability
  • Sharebility

The dataset has not as goal to

  • replace social/community websites and platforms, such as Kitsu, MyAnimeList, and AniDB. The dataset allows to talk about a uniquely identifiable Anime, regardless of the application. However, if the aforementioned websites and platforms would incorporate the Linked Data principles, all of them would at any point know about which Anime they are talking and if its the same one, without the need to rely on record-matching to find the same Anime.
  • be the central hub, datastore, or database for all Anime-related data out there. The initiative is concerned with the general metadata information about Anime. User and application-specific information can and should be hosted by the user and/or the used application.
