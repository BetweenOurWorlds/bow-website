Tools

This page includes some tools that use our dataset. They might also serve as examples for when you want to play with the data.

Semantic Query

If you want to query the dataset using a semantic query, then SPARQL is a good option. Below you can find an example query. This query fetches 10 anime series, including their English name and unique IRI. Hit 'execute' to see the result. Pssst! Don't hesitate to create your own query!


Reverse Lookup

If you have an anime's URL of a specific platform, such as Kitsu, you can use the following search feature to get the unique IRI of that anime.
